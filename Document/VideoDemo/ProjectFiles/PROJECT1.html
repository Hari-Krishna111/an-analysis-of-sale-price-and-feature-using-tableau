About Us

Understand Real Estate with Data

The real estate market is constantly evolving, with numerous factors influencing property prices. Understanding these trends can help buyers, sellers, and investors make informed decisions. This project aims to analyze housing market trends by leveraging data visualization techniques in Tableau.

Using the Transformed Housing Data 2 dataset from Kaggle, we explore key attributes that affect home sale prices, including location, square footage, number of bedrooms, and other essential features.

Objectives

  • Analyze the impact of various property attributes on sale prices.
  • Identify patterns and trends in the housing market over time.
  • Create interactive Tableau dashboards for effective data visualization and insights.
  • Provide a user-friendly interface via a Flask-based website for easy access to visualized data.

Whether for potential buyers, real estate professionals, or policymakers, this project provides valuable insights into the housing market to guide data-driven decisions.
